# Typst CV & Cover Letter

Typst-based CV and cover letter template using the [minimal-cv](https://typst.app/universe/package/minimal-cv) package.

## Requirements

- [Typst](https://typst.app/) (`typst compile`)

## Build

```bash
./compile.sh
```

Or manually:

```bash
typst compile main.typ resume.pdf
typst compile cover-letter.typ cover-letter.pdf
```

## Files

- `main.typ` — CV
- `cover-letter.typ` — Cover letter template
- `compile.sh` — Build script
